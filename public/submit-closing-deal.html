3. 제안서

아이디어 제목

마감딜 — 지금 아니면 못 사먹는다

반찬·밀키트·베이커리·샐러드 등 당일 마감 재고를 사장님이 직접 20~40% 할인 등록하면 마감 임박순으로 자동 노출되는 독립 탭 형태의 타임리밋 할인 피처.

제안 배경

  • 지방 할인랭킹 이미지 훼손: 200원 할인만으로 1위 가능 → 신뢰도 하락
  • 할인율이 낮음: 광고비 경쟁 구조에서 실질 할인 감소, 배민·쿠팡이츠 대비 차별화 없음
  • 상위 고착화: 광고비 오래 집행한 업장만 노출 → 신규 입점 기회 없음

제안 내용

  • 핵심 논리: 반찬·밀키트·베이커리·샐러드는 당일 마감 재고 존재 → 폐기비 회피 동기로 20~40% 자발 할인 유도
  • 마감딜 vs 기존 할인랭킹: 할인랭킹은 광고비 입찰·상시 노출, 마감딜은 마감 임박순 자동 노출·수수료 없음
  • 사장님이 마감 할인율 직접 설정 → 최소 기준 충족 시 자동 노출 (반찬 15%+, 베이커리 15%+, 밀키트 20%+)
  • 갑작스럽게 등장하는 한정 혜택 포맷 — 할인랭킹·요타임딜처럼 독립 진입점으로 배치

기대 효과

  • 마감딜 GMV 순증분
  • 재고 소진율 (사장님 수익 보호)
  • 사장님 자발 등록율 (반찬·베이커리·밀키트 카테고리)
  • 지방에서 더 강력하게 작동 (가게 수 적은 지방 = 마감딜 진입장벽 낮음)

실행 방안

  • Phase 1 (3주): 마감딜 탭 + 사장님 등록 UI + 주문 배지 표시
  • Phase 1.5 (2주): 정기예약 등록 기능 (습관적 사용 유도)
  • Phase 2 (법무 검토 후): 단골 알림 + AI 자동 할인율 제안
