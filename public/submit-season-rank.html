아이디어 제목

요기요 시즌 랭킹 — 게이미피케이션 × 혜택탭 광고 수익화

배달 대기 시간을 활용한 구(區) × 카테고리 기반 월간 시즌 랭킹 게임으로 재방문을 유도하고, 게임 횟수 소진 시 혜택탭 광고 시청으로 충전하는 수익 플라이휠 구조.

제안 배경

  • 혜택탭 UV 지속 감소 → 광고 노출 인벤토리 미활용
  • 배달 대기 시간(25~35분)은 사용자가 앱 안에 머무를 유일한 구간이나 콘텐츠 없음
  • 경쟁 플랫폼 대비 요기요만의 리텐션 훅이 부재

제안 내용

  • 수익 플라이휠: 시즌랭킹 게임 → 재방문 빈도 증가 → 혜택탭 UV 회복 → 광고 수익 증대
  • 경쟁 단위: 구(區) × 카테고리 (예: 강남구 치킨 랭킹)
  • 월간 시즌 리셋: Week1 개막 → Week2~3 추격전 → Week4 클라이맥스 → 월말 정산
  • 게임 횟수 제한: 주문 완료 시 자동 5회 지급, 소진 시 혜택탭 광고 시청으로 +3회 충전
  • 주문→배달대기→게임 진입 플로우: 배달 대기 알림 → 게임 배너 → 시즌 랭킹 참여

기대 효과

  • 혜택탭 광고 수익 증가율
  • 혜택탭 UV·체류시간 (목표: 직전 분기 대비 +15%p)
  • 시즌 참여율, 추월알림 후 24h 재방문율
  • 게임 횟수 소진 → 혜택탭 광고 시청 전환율

실행 방안

  • MVP: 지역 기반 익명 랭킹 + 게임 기본 구현
  • 월간 시즌 운영: 추월알림·마감 긴박감 콘텐츠 시스템화
  • Post-MVP: 친구 추가 기능 + 추월알림 강도 단계별 상승
