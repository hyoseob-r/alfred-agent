3. 제안서

아이디어 제목

혜택탭 광고 구좌 신설 + PLAYYO 경쟁 게임 기반 넛징

혜택탭에 임팩트 있는 외부 광고 구좌를 새로 만들고, PLAYYO 경쟁 게임과 적시 넛징으로 유저를 혜택탭으로 유인한다. 게임은 혜택탭 안에서만 — 엔트리포인트를 고정해 광고 노출을 자연스럽게 필수화하는 구조.

제안 배경

  • 혜택탭에 PLAYYO 미니게임이 이미 존재하나, 한 번 플레이 후 다시 열어야 할 이유가 없음 (동기 부재)
  • 결과적으로 혜택탭 UV가 낮고, 기존 광고 인벤토리(AdFit 배너 1개)가 충분히 활용되지 못하는 상태
  • 배달 대기 시간(25~35분)은 유저가 앱에 머무를 수 있는 유일한 구간이나 적절한 유인 부재

제안 내용

  • 혜택탭 내 광고 구좌 신설: 게임 실행 전 전면 광고, 횟수 충전형 보상형 광고, 랭킹 페이지 네이티브 광고
  • 넛징 ①: 주문 완료 시 "배달 기다리는 동안 혜택탭에서 게임 한판" 앱 내 배너 노출
  • 넛징 ②: 경쟁 랭킹 추월 시 푸시 알림 → 혜택탭으로 딥링크
  • 경쟁 구조: 구(區) × 카테고리 단위 시즌 랭킹, 월간 리셋, 추월 알림으로 재방문 훅 생성
  • 신규 게임 제안: 조리런 — 속도감 있는 장애물 회피 게임으로 경쟁 구조에 최적화
  • 게임 기회: 주문 완료 시 +5회 자동 지급 / 친구 랭킹 초대 시 +3회 / 친구추천(신규 가입 유도) 시 +5회 — 요기요 핵심 행동과 연동

기대 효과

  • 혜택탭 광고 수익 증가 (신설 구좌 노출 수 증가)
  • 혜택탭 UV·체류시간 (목표: 직전 분기 대비 +15%p)
  • 주문 완료 넛징 → 혜택탭 전환율
  • 추월 알림 후 24h 재방문율

실행 방안

  • MVP: 지역 기반 익명 시즌 랭킹 + 혜택탭 내 광고 구좌 신설 + 주문 완료 넛징
  • 조리런 신규 게임 개발 및 PLAYYO 추가
  • Post-MVP: 친구 랭킹 기능 + 추월 알림 강도 단계별 상승
